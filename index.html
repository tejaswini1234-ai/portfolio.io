Two Columns
This Snake Game project uses Python's pygame library to create a simple interactive game. The player controls a snake that moves around the screen, trying to eat randomly positioned food blocks. Each time the snake eats food, it grows in length, and the player's score increases. The game ends when the snake collides with the screen boundary or itself. The game loop handles user input, snake movement, collision detection, and score updates.
This Rock-Paper-Scissors game in Python lets a user play against the computer. The user selects rock, paper, or scissors, while the computer randomly chooses one of the three options. The game then compares both choices and declares a winner based on the classic rules (rock beats scissors, scissors beats paper, and paper beats rock). The program also handles invalid inputs and can run multiple rounds. It's a simple demonstration of conditional logic and randomization in Python.
